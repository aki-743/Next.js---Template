# 内容

ボタンのコンポーネントとして使用。

# コンポーネント

|コンポーネント名|説明|
|---|---|
|DefaultDrawer|Drawer|

# Props

## BaseDrawerProps

|props名|必須|型|デフォルト値|説明|
|---|---|---|---|---|
|anchor|必須|left, top, right, bottom||ドラワーを表示する位置|
|open||boolean||表示・非表示状態|
|toggleDrawer||void||表示・非表示の切り替えを行う関数|
|drawaerList||DrawerList||表示するメニューリスト|
